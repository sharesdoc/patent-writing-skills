# Patent Boilerplate Templates

## 技术领域

```
本发明涉及[技术领域]，尤其涉及一种[具体技术方案概述]的方法、装置及系统。
```

## 发明内容·开头

```
为解决上述现有技术中存在的技术问题，本发明提供一种[方法名称]。
```

## 发明内容·多方面格式

```
第一方面，本发明提供一种……方法，包括以下步骤：
  步骤S1：……；
  步骤S2：……；

进一步地，步骤SX中所述……

第二方面，本发明提供一种……装置，包括：
  XX模块，用于……；
  YY模块，用于……；

第三方面，本发明提供一种电子设备，包括处理器和存储器，所述存储器中存储有计算机程序，所述处理器执行所述程序时实现上述方法的步骤。

第四方面，本发明提供一种计算机可读存储介质，其上存储有计算机程序，该计算机程序被处理器执行时实现上述方法的步骤。

与现有技术相比，本发明具有以下有益效果：
（1）……
（2）……
```

## 附图说明·每行格式

```
图X为本发明实施例中……的XXX图；
```

## 具体实施方式·开头

```
为了使本发明的目的、技术方案和优点更加清楚，下面将结合附图和具体实施例对本发明作进一步详细描述。
```

Optional addition after opening:

```
需要说明的是，在不冲突的情况下，本发明中的实施例及实施例中的特征可以相互组合。
```

## 具体实施方式·结尾

```
以上所述仅为本发明的实施例而已，并不用于限制本发明。对于本领域技术人员来说，本发明可以有各种更改和变化。凡在本发明的精神和原理之内所作的任何修改、等同替换、改进等，均应包含在本发明的权利要求范围之内。
```

## 权利要求书

### 独立方法权利要求

```
1. 一种[方法名称]，其特征在于，包括以下步骤：
步骤S1：……；
步骤S2：……；
```

### 从属方法权利要求

```
2. 根据权利要求1所述的方法，其特征在于，步骤SX中所述……
```

### 独立装置权利要求

```
N. 一种[装置名称]，其特征在于，包括：
XX模块，用于……；
YY模块，用于……；
```

### 设备权利要求

```
N+1. 一种电子设备，包括处理器和存储器，所述存储器中存储有计算机程序，其特征在于，所述处理器执行所述程序时实现权利要求1至M任一项所述方法的步骤。
```

### 介质权利要求

```
N+2. 一种计算机可读存储介质，其上存储有计算机程序，其特征在于，该计算机程序被处理器执行时实现权利要求1至M任一项所述方法的步骤。
```

## 背景技术·现有专利分析模板

```
CNXXXXXXXXXB公开了一种[方法名称]，[客观描述其技术方案，2-3句话]。该方法的局限在于[指出局限性，与本发明形成对比]。
```

## 背景技术·检索结论模板

```
经系统检索（检索日期XXXX年XX月，检索范围覆盖CNKI、万方、维普、Espacenet、USPTO、Google Scholar等数据库），以"[关键词1]"结合"[关键词2]"结合"[关键词3]"进行检索，未发现任何将[方法A]应用于[领域B]的专利或论文。
```
